# UiNumberRoll 组件

## 介绍

UiNumberRoll 组件，属性与事件以源码注释为准。

## 代码演示

```vue
<template>
  <ui-number-roll />
</template>
```

## Props

| 参数           | 说明       | 类型             | 默认值 | 可选值 |
| -------------- | ---------- | ---------------- | ------ | ------ |
| value          | 数据值     | string \| number | 0      | -      |
| color          | 字体颜色   | string           | ""     | -      |
| fontSize       | 字体大小   | string \| number | 32     | -      |
| fontWeight     | 字体粗细   | string           | ""     | -      |
| trimZero       | 去除0      | boolean          | true   | -      |
| thousandsSep   | 千位分隔符 | string           | ""     | -      |
| decimalPlaces  | 小数位数   | number           | 2      | -      |
| duration       | 过渡时间   | string \| number | ""     | -      |
| timingFunction | 动画函数   | string           | ""     | -      |
| customClass    | 自定义类名 | string           | ""     | -      |
| customStyle    | 自定义样式 | string \| object | ""     | -      |
