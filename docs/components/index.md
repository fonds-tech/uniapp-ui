# 组件总览

UniApp UI 提供了 80+ 高质量组件，按功能分为以下几类：

## 基础组件

最基础的 UI 元素，用于构建页面的基本结构。

| 组件                                | 说明             |
| ----------------------------------- | ---------------- |
| [Button 按钮](/components/button)   | 常用的操作按钮   |
| [Icon 图标](/components/icon)       | 基于字体的图标集 |
| [Image 图片](/components/image)     | 增强型图片组件   |
| [Text 文本](/components/text)       | 文本显示组件     |
| [Badge 徽标](/components/badge)     | 在右上角展示徽标 |
| [Tag 标签](/components/tag)         | 用于标记和分类   |
| [Loading 加载](/components/loading) | 加载中状态       |
| [Line 分割线](/components/line)     | 分隔内容区域     |
