# 介绍

UniApp UI 是一个基于 Vue 3 + UniApp 的跨端 UI 组件库，提供 80+ 高质量组件，支持 H5、微信小程序、App 等多端运行。

## 特性

- 🚀 **开箱即用** - 80+ 精心设计的 UI 组件，覆盖移动端常见业务场景
- 📦 **按需引入** - 支持 easycom 自动导入，无需手动注册组件
- 🎨 **主题定制** - 基于 CSS 变量的主题系统，轻松实现个性化定制
- 📱 **多端适配** - 一套代码，同时支持 H5、微信小程序、App 等多端运行
- 🔧 **TypeScript** - 使用 TypeScript 编写，提供完整的类型定义
- ⚡ **高性能** - 优化渲染性能，确保流畅的用户体验
